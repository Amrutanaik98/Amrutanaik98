## 📂 Featured Projects

### 🌍 [Climate Intelligence Platform for Extreme Weather Prediction](https://github.com/Amrutanaik98/Climate-Intelligence-Platform-for-Extreme-Weather-Prediction)
- **Stack:** Python · Kafka · Spark Structured Streaming · Airflow · FastAPI · Streamlit · XGBoost · PyTorch · GCP · PostgreSQL
- Real-time weather ingestion from 80+ cities into a Kafka → Spark → medallion data lake → PostgreSQL star schema pipeline, with XGBoost/LSTM models predicting extreme weather events and an LLM layer for natural-language insights.

### 🤖 [RAG Document Intelligence System](https://github.com/Amrutanaik98/RAG-Document-Intelligence-System)
- **Stack:** Python · FastAPI · Streamlit · LangChain · Sentence Transformers · HuggingFace LLMs · Pinecone · Databricks
- End-to-end retrieval-augmented generation system: document ingestion and chunking, vector embeddings into Pinecone, hybrid semantic/keyword search, and context-aware LLM responses served through a FastAPI backend and Streamlit UI.

### 🛫 [Flight Data Analytics Pipeline](https://github.com/Amrutanaik98/Flight_Data_Analysis)
- **Stack:** Python · PySpark · Airflow · Terraform · AWS (S3, EC2, DynamoDB, Lambda, Glue) · Streamlit
- Airflow-orchestrated pipeline ingesting flight data via API, streaming through Lambda/SQS, batch-processing with Glue, and serving delay-prediction models (XGBoost, Random Forest) through a REST API and Streamlit dashboard.

### 🍽️ [Food Inspection – Dallas & Chicago](https://github.com/Amrutanaik98/Food_Inspection-Dallas_Chicago_BusinessIntelligence)
- **Stack:** Azure Data Factory · Databricks (PySpark) · Snowflake · Power BI · SQL
- End-to-end ELT pipeline with medallion architecture, transforming multi-city food inspection data in Databricks and Snowflake, surfaced through interactive Power BI dashboards with automated data-quality checks.
